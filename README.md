# Audisen

### To build
```bash
make usb
make simulation
make autotests
```

### Usb
Work with .txt, .amp, .ams file
```bash
./usb <file_name>
```

### Simulation
Work with .txt, .ams file
```bash
./simulation <file_name>
```

### Autotests
You need to have `bohemian_rhapsody.ams` and `bohemian_rhapsody.ams` file in the build folder, add them to build folder
```bash
./autotests
```
